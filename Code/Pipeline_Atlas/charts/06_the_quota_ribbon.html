Chart 06 · Mechanism Diagnostics

How the cellular automaton decides how much to build.

The CA is not free to grow as it pleases. At each five-year period it receives a quota — a fixed pixel count to place — distributed across five iterations. Each glowing slab is one period's quota stacked on the running total. Slab heights compound at 12.0% per period: the first slab adds 71,468 pixels; the last adds 138,102, almost double. The visual cadence reveals the simulation's only acceleration mechanism.
