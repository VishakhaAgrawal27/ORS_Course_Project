The class that grows by shrinking.

Bare soil — ex-agriculture, vacant lots — has a strange biography. Historically (1990 to 2020), Random Forest classification swings barren area between 257 and 912 km2 as cropping cycles and Landsat's spectral overlap with sparse vegetation come and go. From 2020 onward the cellular automaton drops it almost monotonically from 411 to 144 km2. The contrast is the visual punchline: the model treats barren as a stage cities pass through, not a state they remain in.
