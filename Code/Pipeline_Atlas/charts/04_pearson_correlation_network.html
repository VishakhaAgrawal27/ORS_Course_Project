Chart 04 · Driver Mechanics

The drivers, weighted by how much they tell the same story.

Five glowing nodes — one per MLR predictor — floating in a force-directed network. Edges connect every pair, with thickness proportional to the absolute Pearson correlation and color encoding the sign: red for positive (move together), blue for negative (move opposite). Drag any node to disturb the layout; correlated drivers visibly snap back together. The graph quantifies what the VIF table only summarized: most drivers are essentially independent.
