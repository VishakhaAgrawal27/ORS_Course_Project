Where each square kilometre of 2020 ends up by 2050.

Four left pillars are the 2020 source classes (water, vegetation, barren, built); four right pillars are the 2050 destination classes. Pillar height = total km2. Glowing particle streams flow from each source to each destination, with stream thickness proportional to the flow's km2. The biggest stream you'll see is vegetation pouring into built: 261.6 km2 of green becoming red.
