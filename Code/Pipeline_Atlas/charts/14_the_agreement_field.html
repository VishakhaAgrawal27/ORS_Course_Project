Chart 14 · Scenario Sweep

Where seventeen models agree, where they don't.

A 334×333 height field. Each tower's height counts how many of the 17 scenarios mark that pixel as built by 2030. The visual is sharply bimodal: 44.8% of pixels are unanimously non-built (no towers), and 44.4% are unanimously built (full-height crimson towers). Between them runs the contested ring — the orange & amber zone where the answer depends on which parameter set you chose.
