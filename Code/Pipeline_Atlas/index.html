About this Atlas

Land-cover classification via Random Forest on Landsat 5/7/8 surface-reflectance imagery (1990, 1995, ..., 2020). Built-up futures projected via a Cellular Automaton driven by an MLR transition probability model with five spatial drivers (slope, elevation, population density, distance to city center, distance to major roads) and a Genetic Algorithm-tuned neighborhood weight (omega), inertia (alpha), and quota rate. Validated against held-out 2020 imagery; full pipeline and per-scenario diagnostics in source notebook.

All numeric values cross-checked against the original simulation pipeline. See the per-chart source notes for traceback paths.
